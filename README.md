# Lions Dashboard 🦁

Plataforma integral para la gestión y monitoreo de operaciones de **Lions Sports Media**. Este sistema centraliza la logística, el control de pauta publicitaria y el seguimiento en tiempo real de eventos deportivos.

## 🚀 Funcionalidades Principales
- **Gestión Operativa:** Control detallado de minutos y presencia de marca por partido.
- **Fixture Inteligente:** Sincronización automática de eventos con datos de ligas internacionales.
- **Sistema de Roles:** Acceso segmentado para Administradores, Productores, Operadores y Personal de Clubes.
- **Reportes y Backups:** Generación de informes de actividad y respaldo global de datos operativos.
- **Seguridad Robusta:** Implementación de políticas RLS (Row Level Security) para garantizar la integridad de la información.

## 🛠️ Plan de Desarrollo V1.1
Estamos evolucionando el sistema bajo una hoja de ruta centrada en la eficiencia y la transparencia:

1.  **Saneamiento y Core:** Estabilización de sesiones, mejoras en la conexión y limpieza de políticas de seguridad.
2.  **Agenda Avanzada:** Centralización de fixtures en base de datos local y conversión automática de horarios internacionales.
3.  **Flujo de Trabajo Estricto:** Proceso de aprobación administrado para "congelar" datos operativos y evitar errores manuales.
4.  **Transparencia Pública:** Generación de reportes automáticos y enlaces públicos para revisión de pauta.
5.  **Modernización de Interfaz:** Rediseño estético enfocado en la visualización de métricas críticas y estados operativos.
